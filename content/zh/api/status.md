## 存款交易状态

存款交易状态列表。

| transaction_status | 名称       | 描述                                        |
| ------------------ | ---------- | ------------------------------------------- |
| `1`                | `待处理`   | 交易正在等待处理                            |
| `2`                | `已完成`   | 交易已完成                                  |
| `3`                | `失败`     | 交易失败                                    |
| `4`                | `手动处理` | 如果状态是手动处理，则需要与客服/管理员联系 |
